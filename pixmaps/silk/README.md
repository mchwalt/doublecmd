# Silk icons (toolbar)

These PNG icons are from the **Silk icon set** by Mark James (famfamfam).

- Source: https://github.com/legacy-icons/famfamfam-silk
  (originally http://www.famfamfam.com/lab/icons/silk/)
- License: **Creative Commons Attribution 2.5** (CC BY 2.5) —
  https://creativecommons.org/licenses/by/2.5/
- Author: Mark James

They are used in this fork as the main toolbar icons (referenced from the
toolbar config via `%COMMANDER_PATH%\pixmaps\silk\<name>.png`), replacing the
non-portable, proprietary `C:\totalcmd\WCMICONS.DLL` references.

Only the subset needed by the toolbar is included. The original Silk file names
are kept for traceability. Per CC BY 2.5, attribution to Mark James is required;
this file provides it.
